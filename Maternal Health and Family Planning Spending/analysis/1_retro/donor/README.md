

# Retrospective estimates of development assistance

Retrospective estimates of development assistance for harmful practices (HP).  


## HP

- `harmful_practices/`

For the harmful practices, a new pipeline was built that runs a harmful practice keyword search across input data, standardizes and aggregates the data, and splits regional projects.  

Other data sources were considered including using IATI to track aid from orgs like UNDP and UN Women, but their reporting to IATI is too incomplete.
